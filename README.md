# Samyak React Project

A modern React application built with Vite and TypeScript.

## Features

- Fast development with Vite
- TypeScript support
- Tailwind CSS for styling
- ESLint configuration

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```
